Descriptor Word Streaming Interface

class src.rsxdwstreaming.xdw_streaming.XdwStreaming(if_ip_addr, port, protocol, options)[source]

Initialize the xDW streaming interface to be able to stream xDW data to generator.

Parameters

  • if_ip_addr (str) – IP address of generator interface, e.g. ‘127.0.0.1’

  • port (int) – network port, defaults to 49152

  • protocol (str) – desired protocol, either TCP or UDP, defaults to ‘TCP’

  • options – see socket.setsockopt for more information
