Descriptor Word Streaming Interface

class src.rsxdwstreaming.xdw_streaming.XdwStreaming(if_ip_addr, port, protocol, options)[source]

Initialize the xDW streaming interface to be able to stream xDW data to generator.

Parameters

  • if_ip_addr (str) – IP address of generator interface, e.g. ‘127.0.0.1’

  • port (int) – network port, defaults to 49152

  • protocol (str) – desired protocol, either TCP or UDP, defaults to ‘TCP’

  • options – see socket.setsockopt for more information

class src.rsxdwstreaming.adw.Adw(payload, seg_interrupt, extension, **kwargs)[source]

ARB descriptor word (ADW) class

Parameters

  • payload – ADW payload

  • seg_interrupt (bool) – Flag to determine if segment can be interrupted, defaults to False

  • extension – ADW extension, supported: XdwExtensionUnused, XdwExtensionBurst, defaults to XdwExtensionUnused

Keyword Arguments

  • ignore_xdw – Flag to determine if ADW is ignored

  • m1 – Marker 1

  • m2 – Marker 2

  • m3 – Marker 3

  • freq_offset – Frequency offset in Hertz

  • level_offset – Level offset in dB

  • phase_offset – Phase offset in degree
